ArchivCost User Manual

Automating Management Cost Systems & Intelligence

Introduction

This notebook LLM application is styled as a classical, minimal archive ledger to relieve cost accountants of manual, repetitive cost analysis. It integrates a powerful custom Gemini 2.5 context engine directly on the front-end.

Core Components

  • Document Library (Left Column): Upload CSV, text, JSON, or PDF records. Preloads demo worksheets like activity-based costing allocations and standard variances.
  • Document Viewer (Middle Column): Interactive render panel highlighting cost allocations, ledger items, variance formulas, and anomalies with direct visual markup.
  • AI Chatbot (Right Column): Fed with current document context to answer accounting queries using advanced Gemini logic.

Accounting Operations Supported

  • Variance Auditing: Calculating Price and Usage variances, pinpointing responsible departments (procurement, operations).
  • Overhead Disposition: Determining whether manufacturing overhead was under- or over-applied, allocating differences to Cost of Goods Sold (COGS).
  • Margin Optimization: Recommending reduction paths for fixed and semi-variable expenses based on volume-profit data.
Security Notice: Your Gemini API key is stored strictly on your local browser session and remains private. All AI calculations execute directly via secure client-to-Google Cloud network requests.
